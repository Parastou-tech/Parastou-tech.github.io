## Overview

I developed a comprehensive full-stack web application designed to streamline the management of customer inventory and orders. Leveraging **React** on the frontend and **Express** on the backend, this application efficiently handles the core operations needed for a modern inventory system.

---

## Key Features

### Full-Stack Architecture
- **Frontend (React)**: Built a responsive and intuitive interface that allows users to view and manipulate inventory in real-time.
- **Backend (Express)**: Handled server-side logic and routing, ensuring smooth communication between the client and the database.

### RESTful API Endpoints
- **POST and DELETE**: Implemented core endpoints for creating and deleting orders, letting users easily add or remove items from their records.
- **Query Routes**: Designed specialized routes to filter data based on user-specific requirements, allowing personalized and efficient data retrieval.

### Data Persistence (MongoDB)
- **Scalable Database**: Utilized **MongoDB** for storing inventory items, customer data, and order details, offering robust querying and indexing capabilities.
- **Schema Design**: Structured collections in a way that supports quick lookups for orders and customers, making the app ready for future expansion and additional features.

### Testing with Postman
- **API Reliability**: Conducted thorough testing of each endpoint with **Postman** to confirm proper functionality and reliability.
- **Use Cases**: Covered scenarios like creating orders, deleting orders, and querying data based on user IDs, ensuring all edge cases are handled.
